🔗 Navigation Container

Fungsi:

Mengelola menu navigasi utama website dengan logo perusahaan dan menu dropdown responsif.

Struktur HTML:

<div class="navigation w-nav"> <div class="navigation-wrap"> <a href="index.html" class="logo-link"> <img src="logo.png" class="logo-image"> <div class="text-block-9">PT ELIAZER NAHOR PRATAMA</div> </a> <div class="menu"> <nav class="navigation-items w-nav-menu"> <a href="about.html" class="navigation-item">About</a> <a href="projects.html" class="navigation-item">Project</a> <a href="experience.html" class="navigation-item">Experience</a> <a href="equipment.html" class="navigation-item">Equipment</a> <a href="team.html" class="navigation-item">Team</a> <a href="contact.html" class="navigation-item">Contact</a> </nav> <div class="menu-button w-nav-button"> <img src="menu-icon.png" class="menu-icon"> </div> </div> </div> </div>

Fitur:

  • Responsive design dengan hamburger menu
  • Logo perusahaan dengan nama lengkap
  • 6 menu utama: About, Project, Experience, Equipment, Team, Contact
  • Tombol "Contact us" dengan link email

📢 CTA Container

Fungsi:

Call-to-Action container untuk mengajak pengunjung melakukan tindakan tertentu (kontak, mulai proyek).

Struktur HTML:

<div class="cta-container"> <div class="cta-wrap"> <div class="cta-text"> <h1 class="h2-headingscta">We construct for you</h1> </div> <div class="cta-paragraph"> <p class="paragraphscta">Delivering exceptional construction services...</p> </div> <a href="contact.html" class="button cc-jumbo-button"> <div class="text-block-2">Start Now</div> </a> </div> </div>

CSS Properties:

  • Height: 50vh (desktop), 45vh (tablet), 40vh (mobile)
  • Background: #2c3e50 (dark blue)
  • Color: White text
  • Layout: Centered content with call-to-action button

🦶 Footer Container

Fungsi:

Container untuk informasi footer, kontak, dan detail perusahaan di bagian bawah halaman.

Struktur HTML:

<div class="footer-container"> <div class="footer-wrap"> <p class="paragraph-tiny">PT Eliazer Nahor Pratama</p> </div> </div>

CSS Properties:

  • Width: 80% (desktop), 100% (mobile)
  • Height: 90vh (desktop), auto (mobile)
  • Background: #ffffff (white)
  • Padding: 10px 40px (desktop), 20px (mobile)
  • Display: Flex dengan center alignment

🔧 Equipment Categories Container

Fungsi:

Menampilkan kategori peralatan konstruksi dalam grid layout responsif dengan hover effects.

Struktur HTML:

<div class="equipment-categories-container"> <div class="equipment-categories"> <div class="equipment-category"> <h3>Alat Berat Konstruksi</h3> <p>Excavator, bulldozer, crane...</p> </div> <div class="equipment-category"> <h3>Peralatan Piping</h3> <p>Mesin las, cutting tools...</p> </div> <!-- 6 kategori total --> </div> </div>

CSS Properties:

  • Max-width: 1200px
  • Grid: auto-fit, minmax(300px, 1fr)
  • Gap: 30px
  • Hover Effect: translateY(-5px) + shadow
  • Border-radius: 8px

📄 Content Section Container

Fungsi:

Container utama untuk konten halaman dengan background blur dan transparansi.

CSS Properties:

  • Background: rgba(255, 255, 255, 0.75)
  • Backdrop-filter: blur(6px)
  • Margin: auto auto 100px
  • Overflow: hidden
  • Height: auto

Sub-containers:

  • container-section-2: 70% width, flex layout
  • container-section-3: 80% width, grid layout
  • container-section-5: 100% height
